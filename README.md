# MzansiBuilds

MzansiBuilds is a platform that allows developers to build in public, share their project progress, collaborate with other developers, and celebrate completed projects.

## Features
- Developer registration and login
- Create and manage projects
- Live feed of developer projects
- Comment and collaboration requests
- Milestone updates
- Celebration wall

## Tech Stack
Backend: Python (Flask)
Frontend: HTML, CSS, JavaScript
Database: SQLite
Version Control: Git & GitHub
